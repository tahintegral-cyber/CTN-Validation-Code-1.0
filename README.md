# CTN Validation Package

**Reproducibility package for the Caprock Transmission Number (CTN) framework**

This repository contains the numerical solver, analytical reference
implementations, validation suite, and unit tests accompanying the manuscript:

> Mbah, H. (2026). *The Caprock Transmission Number: A Dimensionless
> Screening Criterion for Diffusion-Dominated Pressure and Thermal Signal
> Propagation in CO2 Storage Caprocks.* International Journal of Greenhouse
> Gas Control.

The package allows the numerical verification reported in Section 3.9 of the
manuscript to be reproduced from scratch with a single command.

Install the dependencies with:

```
pip install -r requirements.txt
```

## Quick start

Run the complete validation suite:

```
python run_all.py
```

This executes all three validation stages, writes CSV results to `results/`,
writes verification figures to `figures/`, and prints a consolidated
pass/fail summary. Typical runtime is about two minutes on a desktop machine.

Run an individual stage:

```
python stage1_terzaghi.py
python stage2_ctn_sweep.py
python stage3_field_sites.py
```

Run the unit-test suite:

```
pytest tests/ -v
```

## What the validation stages verify

**Stage 1 - Terzaghi consolidation benchmark.**
The Crank-Nicolson solver is verified against the analytical Terzaghi
1-D consolidation solution at three dimensionless time factors, and a
grid-convergence study confirms second-order spatial accuracy. Expected
result: maximum pointwise error below 0.001 % and an observed convergence
order of approximately 2.0.

**Stage 2 - CTN sweep verification.**
The closed-form transmission mapping eta(CTN) = erfc[1/(2*sqrt(CTN))]
(manuscript Eq. 15) is verified against the solver across four decades of
CTN. The finite-domain (sealed-overburden) solution is verified against the
Fourier-series result, and the ratio eta_FD / eta_erfc is confirmed to
approach the method-of-images limit of 2 as CTN approaches zero, establishing
Eq. 15 as a conservative lower bound. Expected result: maximum relative error
below 1 % in the well-resolved range.

**Stage 3 - field-site retrospective consistency.**
CTN is computed for four documented CO2 storage sites - In Salah, Snohvit,
Illinois Basin, and CO2CRC Otway - using prior-published parameters only,
with no calibration to observed outcomes. The resulting regime classification
is compared with the reported field response. For In Salah, the poroelastic
CTN places the site at the MODERATE/HIGH boundary; the coupled
thermo-poroelastic term raises CTN into the HIGH regime, consistent with the
observed surface uplift and microseismicity.

---

## Notes on field-site parameters

All site parameters in `ctn/sites.py` are drawn from the independently
published sources cited in the manuscript bibliography. Hydraulic diffusivity
values are reported to three significant figures and reproduce the CTN values
quoted in the manuscript. No parameter has been adjusted to match observed
monitoring outcomes; the CTN classification is computed from prior-published
values alone.
